WURFL Capabilities

  • is_tablet

    This static capability will return TRUE if a device is a tablet computer, regardless of OS. This would be helpfull in tailoring your content to mid size screens.

  • is_smarttv

    This static capability will return TRUE if a device is a SmartTV device like GoogleTV or AppleTV. These devices often have a different input/remote instead of keyboard or touch.

  • brand_name

    This static capability will identify the brand name like Nokia. Sometimes it can help to know the brand of each device so you know what features they support.
