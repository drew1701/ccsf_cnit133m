Ionic

Ionic

  • Older (Cordova)
  • Creates (Hybrid) Native Apps
  • HTML5, CSS, JavaScript

The Ionic framework is one of the older systems and has a large developer and user community. Generally apps are written in HTML5/CSS/JS and then wrapped in the Cordova wrapper to produce a Hybrid/Native app experience. Developers love it because it does not generally require learning any new languages.

https://ionicframework.com/
React Native

React Native

  • Widest Use (Facebook)
  • Creates Native Apps
  • React Library, JavaScript

React Native has the largest adoption across the web. It was originally developed at Facebook and is new but has rapidly grown to be used by many cutting edge online companies. Development is done in JavaScript using the React Library so the learning curve is very simple. React Native produces true native apps and has the support of leading developer groups.

https://reactnative.dev/
Flutter

Flutter

  • Newest (Google)
  • Creates Native Apps
  • Dart (New Language)

Flutter is an exciting new framework developed by Google. Flutter offers some great new features and performance but it requires learning a new language developed by Google called Dart. Flutter is the newest offering on the market but has the backing of one of the largest web companies on the planet. As more and more developers learn Dart, this will be a growing option in the market.

https://flutter.dev/
