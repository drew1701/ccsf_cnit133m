Progressive Web Applications

Progressive Web Applications (PWA) are web based applications that are written with enhancements that give them many of the features often found only in desktop applications. A well written PWA can open quickly even with low network bandwidth and cache files using a Service Worker. By implimenting a manifest file the PWA can also simulate an installed application with a launch icon and window of its own.

The three main points that define a PWA are capable, reliable, and installable. PWA's can create local databases and use offline storage so they are capable, they can also cache files and operate offline for reliability, and a PWA can be installed to the machine with a launch icon so it feels like a stand alone application. Once a PWA is installed on a machine or device it operates in it's own window, separate from the browser so it can be access via ALT-Tab or other application switching methods.
